## 🐛 常见问题速查

| 问题           | 解决方案                              |
| -------------- | ------------------------------------- |
| 数据不更新     | 检查 `setInterval` 刷新间隔           |
| 动画卡顿       | 减少显示数量或关闭不必要的效果        |
| 颜色看不清     | 修改 `color` 字段确保色差足够         |
| 趋势图为空     | 确保 `sparkline` 数组有 10+ 数据点    |
| 模式切换不工作 | 检查 `@click="switchMode(mode)"` 绑定 |
